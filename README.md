# FileShare-MERN-App

#### For Visit deployed Project [Click Here ](https://myfileshareapp.netlify.app)
## Key Features
* Fully Responsive
* 100MB File Limit
* 24Hrs Link Expire Limit

## Programming Languages & Library
* NodeJS For Backend
* ReactJS for Frontend
#### Library and packages
* Express
* Multer for file save
* EJS Template Engine
* NodeMailer
## Database
* MongoDB
